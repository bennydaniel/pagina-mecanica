  Éste es el proyecto de una página web principal o home page desarrollada con html y css para una empresa de mecánica. Consiste de un encabezado con menú de navegación, un slider, tres articulos breves con enlaces a los mismos y un pie de página con información de la empresa asi como también de sus redes sociales.

ESTRUCTURA:
  La estructura de la página consta de cuatro elementos principales: encabezado, slider,
cuerpo y pie de página; las mismas están contenidas dentro de una sección mas grande.

>>>>>>>>Contenedor<<<<<<<<<<<<<
> Encabezado.           
> Slider.               
> Cuerpo.               
> Pie de página.        
>>>>>>>>Fin del contenedor.<<<<

DESARROLLO:
  La estructura de la página está escrita en código HTML, mientras que el formato y
estilos están escritos con código CSS. Voy a describir la manera en la que fui
desarrollando la página:

1) Código base y contenedor:
  El código html comienza con un formato base de HTM5:

<!DOCTYPE html>
<html lang="es-ES">
<head>
   <meta charset="UTF-8">
   <meta http-equiv="X-UA-Compatible" content="IE=edge">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Mecánica</title>
</head>
<body>
   
</body>
</html>

  Dentro de la etiqueta <body> abro la etiqueta <div> con la clase "container", éste es el contenedor para las secciones principales.

<body>
   <div class="container">
       
   </div>
</body>

  A la vez voy escribiendo los estilos para éstos elementos:
> Para toda la pàgina: margin = 0; padding = 0
> Para "body": background: #3a3d47; font-family: verdana;
> Para la clase "container": max-width: 1200px; background: white; margin: 10px auto;

2) Encabezado con logo, menú de navegación y slider:
  Dentro del contenedor principal comienzo la primera sección de la página: el encabezado. Ésta sección está compuesta por el logo de la empresa y un menú de navegación con enlaces a otras páginas de la empresa: Inicio, Nosotros, Servicios, Noticias y Contacto. 
Utilizo la etiqueta <header> para ésta sección y le doy la clase "superior". Dentro de ésta etiqueta coloco dos divisiones con las clases "logo" y "menu". El logo es una imagen diseñada la cual inserto dentro de su división con la etiqueta <img>. Por otra parte, el menú está conformado por una lista <ul> con la clase "opciones" y en cada item de la lista va uno de los cinco enlaces a las secciones de la empresa. Luego continuo con el slider, el mismo va dentro de una etiqueta <section> con la clase "slider" y cuenta con una imagen a la cual le especifico que ocupe el 100% del ancho del contenedor:

<header class="superior">
            <div class="logo">
                <img src="imagenes/logo-blanco.png" alt="logo">
            </div>
            <div class="menu">
                <ul class="opciones">
                    <li><a href="#">INICIO</a></li>
                    <li><a href="#">NOSOTROS</a></li>
                    <li><a href="#">SERVICIOS</a></li>
                    <li><a href="#">NOTICIAS</a></li>
                    <li><a href="#">CONTACTO</a></li>
                </ul>
            </div>
</header>
<section class="slider">
            <img src="imagenes/slider01.jpg" alt="slider" width="100%">
</section>

  El código css para el encabezado define el color de fondo, el tamaño del logo, la separación con los bordes de la sección y la manera en la que se posicionan los elementos. Tambièn define como se posicionan los botones del menú de navegación y el efecto de cambio de color de los mismos al posicionar el cursor sobre éstos:

/* Formato de la sección */
.superior {
    background: #1551a1;
    overflow: hidden;
}

/* Formato del logo */
.logo {
    float: left;
    width: 300px;
    padding: 35px;
}

/* Formato del menú */

.menu {
    float: left;
    padding: 50px;
}
.opciones li{
    float: left;
    list-style: none;
}
.opciones li a{
    color: white;
    padding: 20px;
    text-decoration: none;
    border: 1px solid white;
}
.opciones li a:hover{
    background: #0c3468;
}


3) Cuerpo:

  El titulo de la página está dentro de una etiqueta <h1>, lleva la clase "titulo" y escrita la leyenda "Bienvenidos a tu mecánica de confianza".
  Luego los artículos van dentro de etiquetas <article> y llevan la clase "caja". El titulo de esta clase es devido a que pienso cada artículo como una caja que contiene distintos elementos: la imagen del servicio, el titulo, la descripción y un boton de acceso.
